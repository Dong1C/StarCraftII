## 人类联邦 (Terran)

人类联邦是来自地球的流亡者后裔建立的政权。他们是三个种族中技术最成熟的，依靠科技和灵活战术取胜。

### 概述

| 特性 | 描述 |
|------|------|
| **别名** | 人类、Terran、人类联邦 |
| **特点** | 科技先进、机动性强、战略多样 |
| **主要单位** | 陆战队员、坦克、战列巡洋舰、医疗兵 |
| **玩法风格** | 灵活的战术转换，强大的后期能力 |

### 核心单位

- **陆战队员 (Marine)** — 最基础的步兵单位，灵活多用
- **劫掠者 (Marauder)** — 重型突击单位，克制步兵
- **坦克 (Siege Tank)** — 远程攻城单位，伤害极高
- **医疗兵 (Medivac)** — 运输和治疗的支援单位
- **战列巡洋舰 (Battlecruiser)** — 终极空中单位

### 指挥官

- **吉姆·雷诺 (Jim Raynor)** — 人类联邦的传奇指挥官
- **阿塔尼斯 (Artanis)** — 混合种族（部分星灵）

## 虫族 (Zerg)

虫族是由亚顿虫巢意识统一的生物体种族。他们以数量和快速繁殖著称，追求完美的生物进化。

### 概述

| 特性 | 描述 |
|------|------|
| **别名** | 虫群、Zerg、异虫 |
| **特点** | 数量压制、快速扩张、生物进化 |
| **主要单位** | 跳虫、刺蛇、蟑螂、雷兽、飞蛇 |
| **玩法风格** | 速攻压制，高爆发，多线作战 |

### 核心单位

- **跳虫 (Zergling)** — 最快速的近战单位，繁殖极快
- **刺蛇 (Hydralisk)** — 通用型中型单位
- **蟑螂 (Roach)** — 地面重型单位，可再生
- **雷兽 (Ultralisk)** — 终极地面生物单位
- **飞蛇 (Viper)** — 空中支援单位，消耗敌人

### 领袖

- **刀锋女王莎娜·凯瑞甘 (Sarah Kerrigan)** — 虫群的绝对统治者

## 星灵 (Protoss)

星灵是古老的高等文明，拥有强大的灵能和等离子技术。他们追求荣誉与战术的完美结合。

### 概述

| 特性 | 描述 |
|------|------|
| **别名** | 神族、Protoss、卡拉的子民 |
| **特点** | 灵能强大、部队精炼、荣誉至上 |
| **主要单位** | 狂热者、机械哨兵、巨像、凤凰、航母 |
| **玩法风格** | 质量碾压，高级兵种配合，讲究战术 |

### 核心单位

- **狂热者 (Zealot)** — 近战精英战士，星灵的基础单位
- **机械哨兵 (Sentry)** — 防御和辅助单位，可制造力场
- **追猎者 (Stalker)** — 灵活的远程攻击单位
- **巨像 (Colossus)** — 重型地面单位，范围伤害
- **凤凰 (Phoenix)** — 快速空中单位，可举起敌人
- **航母 (Carrier)** — 终极空中单位，释放拦截机

### 领袖

- **阿塔尼斯 (Artanis)** — 艾尔星灵的执行官
- **泽拉达尔 (Selendis)** — 星灵圣堂武士领袖

## 游戏模式

| 模式 | 说明 |
|------|------|
| **合作模式** | 与AI指挥官协作完成任务 |
| **天梯模式** | 1v1 排名对战 |
| **多人模式** | 2v2、3v3、4v4 组队对战 |
| **执政官模式** | 两人控制同一阵营 |
| **突变模式** | 每周特定规则挑战 |
